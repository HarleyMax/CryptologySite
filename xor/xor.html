Xor Encryption

To encrypt with the XOR cipher, first the message and key must be converted to binary. This is done by, for each character, obtaining the ascii code and converting the ascii code to a binary string of length 8. Then, the binary version of the message is XORed with the binary version of the key. The key is repeated as necessary to match the length of the binary message. This binary result is then converted back to a string. Any non-ascii characters will be denoted as .
