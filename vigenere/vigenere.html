The Vinegere Cipher is essentially a dynamic Caesar Cipher that decides the shift for each character based on a key. It takes a message and a shift. It works by first obtaining the position in the alphabet for all of the characters in the message and in the shift. It then enciphers the message by computing (position-of-message-character + position-of-shift-character) % 26 for each character and replacing the original character with the letter of the new position. The shift is repeated if it is shorter than the message. To decipher you do the same process but subtract the shift letter position insead. For the purposes of this site, any characters that aren't a letter are simply encrypted as themselves, but they do still cause the current shift key to move. The shift key must only contain letters.
