The Vinegere Cipher is essentially a dynamic Caesar Cipher that decides the shift for each character based on a key.

It takes a message and a key.

It works by first obtaining the position in the alphabet for all of the characters in the message and in the key.

It then enciphers the message by computing

(position-of-message-character + position-of-key-character) % 26

for each character and replacing the original character with the letter of the newly calculated position.

The key is repeated if it is shorter than the message.

To decipher you do the same process but subtract the key letter position instead.

For the purposes of this site, any characters in the message that aren't a letter are simply encrypted as themselves, but they do still cause the current position in the key to move. The key must only contain letters.
